BLUE TEAM


★ Tipos de Malware ★


Adware: Es cualquier pieza de software o aplicación que muestra anuncios en el equipo.


Spyware: Es un tipo de software que se instala en un dispositivo y que vigila en secreto la actividad en línea de la víctima, con el objetivo de recopilar información.


Gusano: Se trata de un tipo de software malicioso que se puede replicar a sí mismo para propagarse de forma ilimitada a otros equipos de la red.


Troyano: Es un tipo de software malicioso desarrollado para disfrazarse de software legítimo y benigno, con el fin de obtener acceso a los sistemas de los usuarios objetivo.


Ransomware: Es un tipo de software malicioso diseñado para bloquear el acceso a una o varias partes de un sistema informático. Lo hace encriptando los datos que se encuentren en dichas localizaciones, y pide un rescate por ellos en forma de altas sumas de dinero.


Rogueware: Es un programa malicioso con la apariencia de antivirus o aplicación de seguridad que alerta a los usuarios de un supuesto problema en sus dispositivos que deben corregir, y ofrece una serie de herramientas para mitigar el problema, las cuales en verdad están haciendo todo lo contrario, es decir, infectar el sistema informático.


Keylogger: Es un software que rastrea o registra las teclas pulsadas en el teclado, normalmente de forma encubierta para que el usuario no sepa que sus acciones están siendo vigiladas.


Rootkit: Se trata de un tipo de Malware diseñado para dar a los piratas informáticos acceso y control sobre un dispositivo objetivo. Aunque la mayoría de los Rootkits afectan al software y el sistema operativo, algunos también pueden infectar el hardware y el firmware de un equipo.

★ Herramientas Blue Team ★


EDR (Endpoint Detection and Response): Es una herramienta inteligente que proporciona monitorización y análisis continuo de los dispositivos finales y sus redes. Su finalidad es la de identificar, detectar y prevenir amenazas avanzadas (APT) con mayor facilidad.


XDR (Extended Detection and Response): Es una herramienta de software como servicio (SaaS) que ofrece seguridad holística y optimizada integrando productos de seguridad y datos en soluciones simplificadas.


SIEM (Security Information and Events Manager): Es una solución de seguridad que ayuda a las organizaciones a detectar y analizar amenazas y responder a ellas antes de que afecten las operaciones del negocio.

Los SIEM rastrean toda la actividad de la red a través de todos los usuarios, dispositivos y aplicaciones mejorando drásticamente la transparencia en toda la infraestructura y detectando amenazas, independientemente de dónde se origine el acceso a los recursos y servicios digitales.


IDS (Intrusion Detection System): Es un dispositivo de supervisión pasivo que detecta amenazas potenciales y genera alertas, lo que permite a los analistas de un SOC o a los responsables de respuesta ante incidentes investigar y responder al incidente potencial.


IPS (Intrusion Prevention System): Es un dispositivo que ayuda a las organizaciones a identificar el tráfico malicioso y bloquea de manera proactiva el ingreso de dicho tráfico a su red. A diferencia de un IDS, un IPS es capaz de responder por él mismo ante ciertas amenazas.


WAF (Web Application Firewall): Protege de múltiples ataques al servidor de aplicaciones web en el Backend. La función del WAF es garantizar la seguridad del servidor web mediante el análisis de paquetes de petición HTTP / HTTPS y modelos de tráfico.
