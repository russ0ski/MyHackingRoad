WINDOWS


★ Atajos de Teclado ★


Ctrl + F: Busca una palabra concreta en un PDF / página web / aplicación.


Ctrl + G: Guarda los cambios efectuados en un documento de texto.


Ctrl + S: Guarda los cambios efectuados en un programa / aplicación.


Ctrl + Z: Vuelve un paso atrás respecto de una modificación.


Win + Impr Pant: Hace una captura de pantalla.


Win + L: Bloquea la pantalla del equipo.


Win + R: Abre la ventana Ejecutar de Windows.


Ctrl + F5: Recarga una página web borrando las Cookies.


Ctrl + Shift + Esc: Abre el Administrador de Tareas.


Ctrl + Shift + I: Abre las herramientas para desarrolladores en un navegador.


Win + Der / Izq: Redimensiona por la mitad el programa / página / documento.


Win + Shift + S: Inicia la herramienta de recortes.


Win + Shift + T: Recupera la ventana cerrada accidentalmente.


Alt + Tab: Se desplaza entre las ventanas existentes.
