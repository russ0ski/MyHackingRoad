★ Tipos de Malware ★


Adware: Es cualquier pieza de software o aplicación que muestra anuncios en el equipo.


Spyware: Es un tipo de software que se instala en un dispositivo y que vigila en secreto la actividad en línea de la víctima, con el objetivo de recopilar información.


Gusano: Se trata de un tipo de software malicioso que se puede replicar a sí mismo para propagarse de forma ilimitada a otros equipos de la red.


Troyano: Es un tipo de software malicioso desarrollado para disfrazarse de software legítimo y benigno, con el fin de obtener acceso a los sistemas de los usuarios objetivo.


Ransomware: Es un tipo de software malicioso diseñado para bloquear el acceso a una o varias partes de un sistema informático. Lo hace encriptando los datos que se encuentren en dichas localizaciones, y pide un rescate por ellos en forma de altas sumas de dinero.


Rogueware: Es un programa malicioso con la apariencia de antivirus o aplicación de seguridad que alerta a los usuarios de un supuesto problema en sus dispositivos que deben corregir, y ofrece una serie de herramientas para mitigar el problema, las cuales en verdad están haciendo todo lo contrario, es decir, infectar el sistema informático.


Keylogger: Es un software que rastrea o registra las teclas pulsadas en el teclado, normalmente de forma encubierta para que el usuario no sepa que sus acciones están siendo vigiladas.


Rootkit: Se trata de un tipo de Malware diseñado para dar a los piratas informáticos acceso y control sobre un dispositivo objetivo. Aunque la mayoría de los Rootkits afectan al software y el sistema operativo, algunos también pueden infectar el hardware y el firmware de un equipo.

★ Puertos más comunes y sus Servicios ★


Puerto 21: Es el puerto utilizado por el protocolo FTP (File Transfer Protocol), que se utiliza para la transferencia de archivos entre dispositivos en una red. Su uso principal es para la transferencia de archivos grandes, como imágenes o videos entre otros archivos.


Puerto 22: Es el puerto utilizado por el protocolo SSH (Secure Shell), que se utiliza para la conexión segura a un servidor remoto. Es utilizado por los administradores de sistemas para la gestión de servidores, entre otras aplicaciones con funciones similares.


Puerto 23: Es el puerto utilizado por Telnet, sirve para establecer conexión remotamente con otro equipo por la línea de comandos y controlarlo. Es un protocolo no seguro ya que la autenticación y todo el tráfico de datos se envía sin cifrar.


Puerto 25: Es el puerto utilizado por el protocolo SMTP (Simple Mail Transfer Protocol), que se utiliza para la transferencia de correo electrónico. Se utiliza para poder enviar correos electrónicos desde un cliente de correo electrónico a un servidor de correo electrónico.


Puerto 53: Es el puerto utilizado por el protocolo DNS (Domain Name System), que se utiliza para traducir nombres de dominio en direcciones IP. Es esencial para la navegación por internet hoy en día, ya que permite que los navegadores web accedan a los sitios web utilizando nombres de dominio y no direcciones IP que son más complicadas.


Puerto 80: Es el puerto utilizado por HTTP (Hypertext Transfer Protocol Secure), el protocolo de transferencia de hipertexto que se utiliza para acceder a todas las páginas web. La mayoría de los sitios web lo utilizan para transmitir todo su contenido.


Puerto 110: Es el puerto utilizado por el protocolo POP3 (Post Office Protocol Version 3), que se utiliza para recibir correo electrónico en un cliente de correo electrónico, en combinación con el puerto 25.


Puerto 443: Es el puerto utilizado por HTTPS (Hypertext Transfer Protocol), el protocolo de transferencia de hipertexto seguro. Permite la transmisión segura de datos por toda la red y se utiliza para transacciones en línea como compras, transacciones bancarias, entre otros.


Puerto 445: Es el puerto utilizado por el protocolo de red SMB (Server Message Block), que permite compartir archivos, impresoras, etcétera, entre nodos de una red de computadoras que usan el sistema operativo Microsoft Windows.

★ Atajos de Teclado ★


Ctrl + F: Busca una palabra concreta en un PDF / página web / aplicación.


Ctrl + G: Guarda los cambios efectuados en un documento de texto.


Ctrl + S: Guarda los cambios efectuados en un programa / aplicación.


Ctrl + Z: Vuelve un paso atrás respecto de una modificación.


Win + Impr Pant: Hace una captura de pantalla.


Win + L: Bloquea la pantalla del equipo.


Win + R: Abre la ventana Ejecutar de Windows.


Ctrl + F5: Recarga una página web borrando las Cookies.


Ctrl + Shift + Esc: Abre el Administrador de Tareas.


Ctrl + Shift + I: Abre las herramientas para desarrolladores en un navegador.


Win + Der / Izq: Redimensiona por la mitad el programa / página / documento.


Win + Shift + S: Inicia la herramienta de recortes.


Win + Shift + T: Recupera la ventana cerrada accidentalmente.


Alt + Tab: Se desplaza entre las ventanas existentes.
