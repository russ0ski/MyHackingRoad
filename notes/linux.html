★ Comandos BASH ★


sudo su: Nos permite acceder a los privilegios de super usuario.


man + comando: Nos abre un manual con toda la información del comando solicitado.


comando --help: Nos muestra todos los parámetros disponibles para el comando solicitado.


whoami: Nos informa sobre qué usuario está actualmente activo en la sesión.


alias + nombre='comando': Permite cambiar la sintaxis de un comando por el que indiquemos.


comando1 | comando2: Permite la ejecución de 2 o más comandos simultáneamente.


cd + ruta: Te desplaza por todo el árbol de directorios del sistema.


pwd: Nos muestra en qué posición del árbol de directorios estamos.


ls: Muestra una lista del contenido que se encuentra en el directorio actual.


ls -l: Muestra una lista del contenido con sus respectivos permisos.


clear: Limpia de contenido la línea de comandos.


echo + texto: Imprime por pantalla el texto que escribamos previamente.


touch + fichero: Crea un fichero vacío en el directorio actual.


nano + fichero: Nos permite editar el fichero seleccionado.


cat + fichero: Nos permite ver el contenido del fichero.


cat -n + fichero: Nos permite ver el contenido del fichero con cada línea enumerada.


cat + fichero1 > fichero2: Permite volcar el contenido de un fichero a otro que creemos.


sort + fichero: Nos permite ver el contenido del fichero ordenado alfabéticamente.


sort -n + fichero: Nos permite ver el contenido del fichero ordenado numéricamente.


grep + palabra + fichero: Nos busca una palabra específica del fichero que indiquemos.
